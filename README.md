# Não Seja o Imprudente

SPA em React + Vite sobre acidentes em rodovias federais brasileiras, com foco em conscientização no trânsito.

O projeto usa dados da Polícia Rodoviária Federal (PRF) e deixa explícito em toda a narrativa que o recorte cobre apenas ocorrências registradas nas BRs entre 2023, 2024 e 2025. Não representa todo o trânsito do Brasil.

## Rodar localmente

```bash
npm.cmd run dev
```

## Build

```bash
npm.cmd run build
npm.cmd run preview
```

## Estrutura

- `src/components/`: blocos reutilizáveis da interface
- `src/sections/`: seções da narrativa principal
- `src/data/`: conteúdo e dados usados no site
- `src/hooks/`: hooks utilitários
- `src/utils/`: formatação e helpers
